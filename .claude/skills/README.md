## Included skills

| Skill | Purpose |
| --- | --- |
| brainstorming | Turn ideas into fully formed designs through collaborative questioning |
| dispatching-parallel-agents | Parallelize independent work with subagents |
| executing-plans | Execute written implementation plans in batches with review checkpoints |
| finishing-a-development-branch | Wrap up a branch: verify, merge/PR, clean up |
| receiving-code-review | Respond to code review feedback with technical judgment |
| requesting-code-review | Get code reviewed before merging |
| subagent-driven-development | Drive a plan with fresh subagents per task |
| systematic-debugging | Root-cause analysis before fixing (four-phase debugging) |
| test-driven-development | RED-GREEN-REFACTOR discipline |
| using-git-worktrees | Isolated workspaces for parallel/risky work |
| using-superpowers | How to find and use these skills |
| verification-before-completion | Verify work actually passes before claiming done |
| writing-plans | Write detailed implementation plans |
| writing-skills | Create/edit skills following best practices |

To update, re-copy the `skills/` directory from the upstream repository.
